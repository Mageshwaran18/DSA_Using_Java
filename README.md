## Data Structure & Algorithm using JAVA by Strivers Sheet

| Concept                     | Formula                                | Excludes Empty? |
| --------------------------- | -------------------------------------- | --------------- |
| Permutations of full array  | $N!$                                   | Yes             |
| Substrings (contiguous)     | $\frac{N(N+1)}{2}$                     | No              |
| Subsequences                | $2^N$ (or $2^N - 1$)                   | Choose          |
